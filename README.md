# tool_web这是一款小工具集合

## 项目运行，先安装相关的包
```
npm install
```

### 在开发模式下启动
```
npm run serve
```

### 打包编译
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
